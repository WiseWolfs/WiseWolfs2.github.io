node-inspector调试Node.js应用

  • node.js
node.js

用了几天的Node.js，每次出现错误，都是通过console.log这种方式来排错，感觉效率很低，还不易调试。比较几种调试方法，觉得node-inpspector最为方便。

1.使用npm进行全局安装。

2.启动inspector服务

3.debug模式运行node.js应用

4.通过URL http://127.0.0.1:8080/debug?port=5858 来访问

ps:8080是默认的端口，可以通过node-inspector –web-port = {port}来修改。
